Highbrow: A Backbone Base library

Or, a library with "intellectual or cultural pretensions; intellectual snob."

Highbrow is an extension of Backbone extracted from a working application. It also includes a router based off page.js, and is adapted to prerender the application on the server.

The goal is a fast responsive, application that feels like it belongs on the web.
